Слайд 2 / 11 — Аннотация

Аннотация

Разработана платформа для обработки экспериментальных данных в распределённой среде. Реализован метод параметризованного приближения целевой функции. Архитектура включает ASP.NET Core, React, Docker, Kubernetes. Эксперименты показали сокращение времени обработки на 33% при сохранении точности.

Реализация

  • Метод деформируемого многогранника (Нелдера — Мида)
  • Backend: ASP.NET Core
  • Frontend: React SPA
  • Оркестрация: Kubernetes

Ключевые результаты

  • Время обработки: −33%
  • Потребление ОЗУ: 1220 → 1040 МБ
  • RSD: 1,6% (без изменений)
  • C# vs Python: 11,4× быстрее
Слайд 3 / 11 — Введение

Введение

Автоматизация обработки экспериментальных данных в распределённых исследовательских центрах требует обеспечения воспроизводимости вычислений и масштабируемости ресурсов.

Цель работы

Формализовать модель оркестрации вычислительных процессов в среде Kubernetes и подтвердить её эффективность в условиях межлабораторных сличительных испытаний.

Проблема

Разнородность методов, ручное управление, неравномерная загрузка серверов

Решение

Единая платформа с контейнеризацией и оркестрацией в Kubernetes

Слайд 4 / 11 — Формализация

Формализация задачи полиэдрального метода

Целевая функция

S(x) = Σi=13 (1 − Ri(x) / Ai)² / 10 → min

Нормировка

Ri(x) ⇒ Ri(x) · 10−6

Вектор параметров

x = (X1, X2, X3, X4)T
Xi ∈ [xmin; xmax], i = 1..n

Обозначения и параметры

Ai

Экспериментальные данные — значения, полученные непосредственно из эксперимента (скорости химических реакций)

Ri(x)

Расчётные значения — рассчитанные значения скоростей реакций на основе текущих параметров модели x

Требуют нормировки: умножение на 10−6

x = (X1, ..., X4)T

Вектор параметров — искомые параметры модели в четырёхмерном пространстве, каждый из которых ограничен диапазоном [xmin; xmax]

S(x) → min

Целевая функция — сумма квадратов относительных отклонений расчётных значений от экспериментальных. Минимизация обеспечивает наилучшее соответствие модели данным

Слайд 5 / 11 — Целевая функция

Метод параметризованного приближения: поиск минимума целевой функции

Задача

Минимизация суммы квадратов относительных отклонений:

S(x) = Σ (1 − R(x)/A)²

Алгоритм

Метод деформируемого многогранника (Нелдера — Мида). Много итераций, чувствителен к задержкам.

Реализация

C#/.NET в Docker-контейнере

Слайд 6 / 11 — Алгоритм

Процедура отыскания минимума S(x)

1

Отражение

xn+3 = xn+2 + α · (xn+2 − xh)

α = 1

2

Растяжение

xn+4 = xn+2 + β · (xn+3 − xn+2)

β = 2,8

3

Сжатие

xn+5 = xn+2 + γ · (xh − xn+2)

γ = 0,6

4

Редукция

xi(k+1) = (xi + xl) / 2

Вдвое от xl

Критерий окончания

((1 / (n + 1)) · Σi=1n+1 (S(xi(k)) − S(x̄))²) < ε

где x̄ — значение целевой функции в центре тяжести
